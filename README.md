# Анализ сервиса Яндекс.Афиша

## Описание проекта

Исследовательский и продуктовый анализ данных сервиса Яндекс.Афиша с целью оценки пользовательского поведения, эффективности маркетинга и окупаемости каналов привлечения.

В рамках проекта выполнен анализ ключевых метрик продукта и подготовлены рекомендации по оптимизации маркетинговых расходов.

---

## Цели исследования

- Изучить активность пользователей сервиса  
- Оценить удержание пользователей (Retention)  
- Рассчитать стоимость привлечения клиентов (CAC)  
- Оценить пожизненную ценность клиента (LTV)  
- Проанализировать окупаемость маркетинговых каналов (ROI)  
- Выявить наиболее эффективные источники трафика  

---

## Используемый стек

- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Когортный анализ  
- Юнит-экономика  

---

## Выполненные задачи

- Подготовка и очистка данных  
- Исследовательский анализ данных (EDA)  
- Расчет DAU / WAU / MAU  
- Анализ Retention Rate  
- Построение когорт пользователей  
- Расчет CAC / LTV / ROI  
- Сравнение рекламных каналов  
- Формирование бизнес-рекомендаций  

---

## Основные выводы

- Часть рекламных каналов имеет низкую окупаемость  
- Повторные пользователи формируют более высокий LTV  
- Эффективность каналов существенно различается  
- Возможна оптимизация маркетингового бюджета за счет перераспределения расходов  

---

## Практическая ценность проекта

Проект демонстрирует навыки продуктовой аналитики, работы с метриками, принятия решений на основе данных и понимание экономики цифрового продукта.
